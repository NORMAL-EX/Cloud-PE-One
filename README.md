# 使用
包管理器：npm

# 开发服务器
```batch
rem 启动渲染进程调试
npm run dev 

rem 启动主进程调试
npm run tauri:dev

rem 一键启动
cmd /c "start npm run tauri:dev"
```

## 构建
直接输出二进制可执行文件
```batch
npm run tauri build
```
